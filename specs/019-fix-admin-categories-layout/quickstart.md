# Quickstart

This feature is a set of UI bug fixes and layout improvements for the Admin Panel. There is no new functionality to start using.

To verify the changes:
1. Navigate to `/admin/categories`.
2. Ensure you have categories with subcategories.
3. Click "Subcategories" on different parent categories and verify only the correct children are shown.
4. Resize the window or use a mobile device and verify the sidebar collapses into a hamburger menu.
5. On desktop, scroll down a long list and verify the sidebar and admin header remain fixed/sticky while only the content list scrolls.
